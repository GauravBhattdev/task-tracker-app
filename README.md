# Task Tracker App

# Description
A responsive  task management web application built  using HTML , CSS and JavaScript.

# Features 

### Add Task 
Users can dynamically add tasks  to the tracker.
# You can add only task at one time

### Delete Task
Tasks can be removed individually.

### Mark as Completed
Completed tasks are visually differentiated using strike-through styling.

### Local Storage
Tasks are stored in browser local storage and persist after refresh.

### Dark Mode
Users can switch between light and dark themes.

### Search Feature
Tasks can be searched in real-time.

### Task Priority
Users can assign Low, Medium, or High priority labels.

### Toast Notifications
Notifications appear after actions like add, delete, or edit.

### Responsive Design
The layout adapts properly for mobile and desktop screens.
